Semana 3

Descripción

Se exploraron las librerías Bootstrap 5 y Tailwind CSS. Bootstrap facilita el uso de componentes prediseñados, mientras que Tailwind permite un control preciso mediante clases utilitarias.

Ejercicios de laboratorio - Resultados

Se desarrollaron páginas con ambas librerías, aplicando layouts, estilos y componentes. Se comparó la eficiencia entre ambas metodologías.

Reflexión

¿Qué aprendí?: A crear interfaces modernas con herramientas populares del frontend.
¿Cómo aprendí?: A través de prácticas guiadas, documentación oficial y discusión colaborativa.
