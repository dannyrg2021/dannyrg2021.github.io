Semana 13

Descripción

Diseño de APIs RESTful: Se presentó el paradigma REST, el uso de APIs de Hipermedios y conceptos como HATEOAS. Se serializaron modelos y fuentes externas de datos.

También se aplicaron técnicas de filtrado, paginación, control de acceso y límites de uso. Se utilizaron ruteadores, conjuntos de vistas, protección CSRF, llamadas AJAX y control CORS.

Ejercicios de laboratorio - Resultados

Se desarrolló un sistema backend con una API RESTful funcional, permitiendo solicitudes GET, POST, PUT y DELETE, incluyendo filtros y paginación.

Reflexión

¿Qué aprendí?: A diseñar e implementar una API RESTful moderna y segura.
¿Cómo aprendí?: Mediante práctica intensiva, documentación y pruebas con herramientas como Postman.
