Semana 9

Descripción

Tecnología Web Backend: Se revisó la arquitectura de aplicaciones web, el funcionamiento de los servidores web y la lógica detrás del procesamiento del lado del servidor. Se introdujeron conceptos fundamentales sobre lenguajes y frameworks para backend.

Desarrollo de Aplicaciones Web: Se analizaron tecnologías clave como PHP y JSP, enfocándose en su rol para crear aplicaciones web dinámicas y conectadas a base de datos.

Ejercicios de laboratorio - Resultados

En el laboratorio se realizó el despliegue de aplicaciones web backend utilizando tanto PHP como JSP. Se experimentó con servidores locales y configuración básica para comprobar su funcionamiento.

Reflexión

¿Qué aprendí?: A comprender cómo el backend responde a las solicitudes del cliente, así como a implementar aplicaciones funcionales con PHP y JSP.
¿Cómo aprendí?: A través de teoría explicativa, ejercicios de práctica guiada y despliegue real en entorno de servidor.
