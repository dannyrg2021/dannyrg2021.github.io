Semana 1

Descripción

Durante esta semana se abordaron los fundamentos de las tecnologías web. Se exploraron conceptos clave como sistema web, aplicación web, sitio web y página web, según la clasificación propuesta por Tim Berners-Lee (W3C, 2023). También se revisó el funcionamiento del internet mediante el modelo cliente-servidor y protocolos como DNS, TCP/IP y HTTP (RFC 2616). Se identificaron los roles esenciales en el desarrollo: frontend, backend y fullstack.

Ejercicios de laboratorio - Resultados

Se instaló Visual Studio Code (VSC), configurando extensiones como Live Server, Prettier y GitLens. Se practicaron atajos de teclado y navegación por la interfaz. Fuente: Microsoft Docs.

Reflexión

¿Qué aprendí?: A diferenciar tipos de soluciones web y el funcionamiento del modelo cliente-servidor.
¿Cómo aprendí?: Mediante exposiciones, uso del aula virtual, VSC y documentación oficial (W3C, Microsoft).
