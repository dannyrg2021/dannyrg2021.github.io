Semana 11

Descripción

Introducción a Django: Se presentó el patrón de arquitectura MVC y su adaptación como MTV en Django. Se realizó la instalación del framework, gestión inicial de proyectos y su integración con servidores como Apache.

Estructura y vistas: Se generó la estructura de un proyecto desde línea de comandos. Se trabajó con vistas basadas en funciones y clases, y se configuraron rutas mediante URLs.

Plantillas: Se utilizaron plantillas HTML en Django, aplicando herencia, bloques, etiquetas y filtros para generar contenido dinámico.

Modelo de datos: Se construyeron modelos de base de datos con sus respectivos campos y se exploró el uso del ORM (API para base de datos).

Ejercicios de laboratorio - Resultados

En el laboratorio se creó un proyecto Django funcional, con modelos conectados a una base de datos, rutas configuradas y vistas enlazadas con plantillas. Se implementaron funcionalidades básicas de una aplicación web completa.

Reflexión

¿Qué aprendí?: A estructurar aplicaciones completas con Django, usando MVC adaptado a MTV, plantillas, vistas y modelos.
¿Cómo aprendí?: A través de prácticas guiadas, documentación oficial y desarrollo paso a paso en laboratorio.
